# 登录

## 登录名

口令文件(/etc/passwd)中的登录项由7个部分组成，分别是：登录名、加密口令、用户ID、组ID、注释字段、home目录、shell程序。

系统中的加密口令已经保存在其他文件中。

## shell

从`/etc/passwd`文件的最后一个字段可以查看登录用户使用的shell。

常用的shell有如下：

* Bourne shell `/bin/sh`
* Bourne-again shell `/bin/bash`
* C shell `/bin/csh`
* Korn shell `/bin/ksh`
* TENEX C shell `/bin/tcsh`

**NB**：在Linux上，`/bin/sh`链接到`/bin/bash`。
